# 🚀 Maspenx WhatsApp Gateway PRO

Fitur lengkap:
- ✅ Simpan media otomatis
- ✅ Autentikasi token
- ✅ Webhook N8N
- ✅ Forward ke Telegram

## 📦 Jalankan

```bash
docker-compose up -d
```

### 🧠 Lingkungan (.env)
- WEBHOOK_URL
- TELEGRAM_BOT_TOKEN
- TELEGRAM_CHAT_ID

❗ Anda belum memiliki Bot Telegram atau N8N Endpoint, konfigurasi ini bisa disesuaikan nanti.
